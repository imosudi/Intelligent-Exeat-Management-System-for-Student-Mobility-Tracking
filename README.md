## Background Studies
### EXEAT System
A formal system used in educational institutions to authorise student movement. This typically requires multiple approvals from administrators, parents, and security.

### Agent-Based Modelling (ABM)
A simulation method where individuals (agents) interact under predefined rules. In this context, agents are students, administrators, security personnel, and parents.

### System Dynamics
A modelling technique used to simulate complex systems by focusing on feedback loops and time delays. This method helps in understanding how system behaviour evolves over time.

### Role-Based Access Control (RBAC)
A system used in simulations to assign specific actions and permissions to users based on their roles, improving system security and simplifying decision-making.

### State Transitions
The changes an agent undergoes from one state to another (e.g., from "waiting" to "approved" in the EXEAT process), crucial for simulating realistic workflows.

### Notification Simulation
Simulating the real-world notification system to alert agents (e.g., students, parents) about the status of EXEAT requests.

### Mobility Pattern Modelling
The study of how students move within the institution, influenced by approvals and other constraints, providing insights into operational efficiencies and security considerations.
